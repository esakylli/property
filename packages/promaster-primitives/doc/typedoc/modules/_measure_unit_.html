UnitConverter

UnitConverter: OffsetConverter | CompoundConverter | FactorConverter | IdentityConverter
  • Defined in measure/unit.ts:109

This type represents a converter of numeric values. Sub-types must ensure unicity of the identity converter. In other words, if the result of an operation is equivalent to the identity converter, then the unique IDENTITY instance should be returned.
